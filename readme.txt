# smart-city

This project allows a user view to mapped out auto-generated pollution levels of Belfast City against api overlays including google maps and NI bike infrastructure.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Windows:


Install Python 3. (https://www.python.org)


Mac:


Install Python 3. (https://www.python.org/download/releases/3.0/)


Windows:


1. Open up Command Prompt and navigate to the SmartCity/WINDOWS/ folder.

2. Run this command: 'python -m http.server --cgi 8000' (minus the single-quotes)

3. Open an internet browser and head to http://localhost:8000
5. Explore the system.



Mac:


1. Open up a Terminal window and navigate to the SmartCity/MACOSX/ folder.

2. Run this command: 'python3 -m http.server --cgi 8000' (minus the single-quotes)

3. Open an internet browser and head to http://localhost:8000
5. Explore the system.
